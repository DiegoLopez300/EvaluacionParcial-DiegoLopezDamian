# Aplicaciones Móviles - Diego Lopez

## Descripcion
El presente campus virtual está enfocado en el curso de Aplicaciones Móviles. Su funcionamiento se basa en el ingreso del código de estudiante, el cual permite acceder al contenido y a las funcionalidades correspondientes.
La aplicación presenta una interfaz simple, intuitiva y fácil de usar, diseñada para que cualquier usuario pueda navegar de manera clara y eficiente.
Una vez se corrobore que el usuario ingresado existe y/o es correcto el estudiante tendra acceso y podra vizualizar sus notas y horarios

## Plantilla Blanca
En esta sección se desarrolló toda la estructura principal del proyecto. Primero, se analizó el espacio disponible en la pantalla con el fin de organizar adecuadamente los elementos visuales.
Posteriormente, se planificó la distribución de los componentes de la aplicación, asegurando una correcta disposición que permita una navegación clara, funcional y estéticamente ordenada.
